# Trunk Compress 🐘

> A software engineering project focused on Lossless Data Compression and Entropy Analysis.

## Objective
To develop a robust Command Line Interface (CLI) tool capable of reducing text file sizes through entropy-based algorithms, ensuring 100% data integrity (Lossless).

## Tech Stack
- **Runtime:** Node.js (LTS)
- **Language:** JavaScript (ES6+)
- **Core:** Streams & Buffers API

## Roadmap (Phase 1)
- [ ] Data flow architecture definition.
- [ ] Implementation of RLE (Run-Length Encoding) algorithm.
- [ ] Command Line Interface (CLI) setup.
- [ ] Bit-level manipulation for optimized storage.
